# Air India Operations Performance Dashboard
### Power BI Dashboard using Official DGCA Data

## Project Overview
Monthly operations performance dashboard for Air India built using 
official DGCA (Directorate General of Civil Aviation) data. Analyses 
2017 flight operations including passenger traffic, load factor, 
aircraft departures and cargo performance.

## Key Findings
- **Average Load Factor: 84.88%** — above industry healthy benchmark 
  of 80-85%, indicating strong seat utilisation
- **February** recorded peak passenger volumes — driven by peak winter 
  travel season and corporate travel resumption
- **August** showed lowest departures — consistent with seasonal 
  travel dip patterns in Indian aviation

## Tools Used
- Power BI Desktop — dashboard development
- Power Query — data cleaning and transformation
- DAX — custom measures and KPI calculations
- Official DGCA Data — government source, highly credible

## Files in this Repository
- `air_india_dgca_2017.csv` — official DGCA monthly operations data

## Dashboard Visuals
- KPI Cards — Total Passengers, Average Load Factor, Total Departures
- Bar Chart — Passengers Carried by Month
- Line Chart — Load Factor Trend across 12 months
- Combo Chart — Departures vs Passengers monthly comparison
- Interactive Slicer — filter by Month

## Data Source
Official DGCA Air Traffic Statistics — published by Ministry of Civil 
Aviation, Government of India.
Data portal: data.gov.in
